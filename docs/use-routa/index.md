# Use Routa

This section explains how to choose the right working mode after Routa is installed and one
provider is available.

## Working Modes

| Mode | Best for | Recommendation |
| --- | --- | --- |
| [Sessions](/use-routa/sessions) | starting work quickly, recovery, one main thread | Default |
| [Kanban](/use-routa/kanban) | repeatable delivery flow, lane automation, review gates | Process-driven |
| [Team](/use-routa/team) | multi-specialist execution and explicit coordination | Coordination-driven |

## Recommended Order

1. Start with [Sessions](/use-routa/sessions) unless you already know you need workflow
   automation or lead-driven delegation.
2. Move to [Kanban](/use-routa/kanban) when you want delivery stages and visible card flow.
3. Use [Team](/use-routa/team) when the coordination problem itself is first-class.

## What This Section Covers
