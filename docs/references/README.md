# References

This directory is reserved for LLM-friendly reference material.

Use it for:
- distilled external references that agents need frequently
- stable internal reference sheets for protocols, frameworks, or tooling
- machine-scannable summaries that are easier to consume than vendor docs

Do not use it for:
- design decisions specific to Routa
- work-in-progress plans
- incident reports

Suggested first additions:
- `acp-llms.txt`
- `tauri-llms.txt`
- `drizzle-llms.txt`

Project-specific references:
- `../specialists/README.md`: 从 `resources/specialists/**/*.yaml` 自动生成的 specialist 总览与独立页面入口
- `office-document-viewer-wasm-reader/`: Office 文档 WASM reader 的逆向分析、参考实现边界和产品化拆分建议
